JUMP START A BATTERY
Steps:
1. Position donor vehicle close. Both engines off. Both in park.
2. Red clamp → dead battery positive (+) terminal.
3. Other red clamp → donor battery positive (+) terminal.
4. Black clamp → donor battery negative (−) terminal.
5. Other black clamp → unpainted metal on dead car's engine block (ground). NOT the dead battery negative.
6. Start donor vehicle. Wait 2-3 min.
7. Start dead vehicle. If no crank, wait 5 more min.
8. Remove clamps in reverse order (ground first, donor negative, donor positive, dead positive).
9. Drive revived vehicle 20+ min to recharge. Or connect a charger.
Notes:
- Wrong polarity fries ECUs. Double-check red-to-positive before starting.
- Swollen or cracked battery: do not jump. Replace it.
- If it needs jumping twice in a week, battery or alternator is failing.
