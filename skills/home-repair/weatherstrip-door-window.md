WEATHERSTRIP A DOOR OR WINDOW

Steps — Door:
1. Close the door. Look for daylight around the edges. Run your hand along the frame to feel drafts. Hold a lighter or incense stick near the edges — movement indicates leaks.
2. Remove old weatherstripping by peeling or prying it off. Clean the surface with rubbing alcohol.
3. Measure each side of the frame. Cut new weatherstripping to length.
4. For the top and sides: adhesive-backed foam tape (for small gaps) or V-strip/tension seal (for larger gaps). Peel backing and press into the door stop so the door compresses it when closed.
5. For the bottom: install a door sweep. Screw it to the inside of the door so the sweep just touches the threshold when closed.
6. Close the door. Check for remaining drafts. Adjust sweep height if needed.

Steps — Window:
1. Clean the window channel/sash with a brush and rubbing alcohol.
2. For double-hung windows: apply V-strip into the channel where the sash slides. It should create friction without binding the window.
3. For gaps between sash and frame: apply foam tape to the frame where the window meets it when closed.
4. Close the window. It should compress the weatherstripping without being hard to close.

Notes:
- Foam tape is cheapest but wears fastest. V-strip and silicone-filled tube last longer.
- If the door frame is warped or the window frame is rotted, weatherstripping won't seal the gap. Fix the frame first.
- For very old windows, rope caulk (removable) is a quick seasonal fix.
