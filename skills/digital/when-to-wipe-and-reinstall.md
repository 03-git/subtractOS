WHEN TO WIPE AND REINSTALL (DECISION TREE)

Steps:
1. TRY THESE FIRST (most "slow computer" problems don't need a reinstall):
   - Restart the computer (actually restart, not just close the lid).
   - Check available storage — if drive is 90%+ full, free up space.
   - Check what's running at startup and disable unnecessary programs.
   - Run a malware scan (Malwarebytes free).
   - Update your OS and drivers.
   - If browser is slow: clear cache, disable extensions, check for rogue extensions.

2. REINSTALL IF:
   - Malware scan finds something it cannot remove.
   - You gave remote access to a scammer.
   - System is unstable after a botched update or driver installation.
   - Performance problems persist after all basic troubleshooting.
   - You're selling or giving away the computer.
   - You want a clean start (years of accumulated software bloat).

3. BEFORE REINSTALLING:
   - Back up all personal files to an external drive or cloud storage. Verify the backup works.
   - List software you need to reinstall. Check you have license keys.
   - Export browser bookmarks and password manager data.
   - Note your wifi password.

4. REINSTALL:
   - Mac: Restart, hold Cmd+R (Intel) or hold power button (Apple Silicon) to enter Recovery. Erase disk, reinstall macOS.
   - Windows: Settings > System > Recovery > Reset this PC > Remove everything.
   - Linux: Boot from USB installer. Format and reinstall.

Notes:
- A reinstall fixes software problems only. If hardware is failing (clicking hard drive, overheating, screen artifacts), a reinstall won't help — get the hardware diagnosed.
- After reinstalling: install updates, set up backups, install a password manager, then restore your files.
