REVIEW APP PERMISSIONS (PHONE)

Steps:
1. IPHONE:
   - Settings > Privacy & Security.
   - Review each category: Location Services, Camera, Microphone, Contacts, Photos, Tracking.
   - For each: see which apps have access. Revoke any that don't need it for core function.
   - Settings > Privacy & Security > Tracking: turn off "Allow Apps to Request to Track."
   - Settings > Privacy & Security > Location Services: set apps to "While Using" instead of "Always" where possible.

2. ANDROID:
   - Settings > Privacy > Permission Manager.
   - Review each category: Location, Camera, Microphone, Contacts, Files, Phone.
   - For each: see which apps have access. Revoke unnecessary permissions.
   - Settings > Privacy > Ads: opt out of ad personalization.
   - Settings > Location: set apps to "Allow only while using the app."

3. FOR BOTH: Ask yourself for each permission: "Does this app need this to do what I use it for?" A flashlight app does not need your contacts. A calculator does not need your location.
4. Repeat this review every 3-6 months. Permissions can change after app updates.

Notes:
- Revoking a permission won't delete data the app already collected.
- If an app stops working after revoking a permission, re-grant it only if the permission makes sense for that function. If it doesn't, find an alternative app.
