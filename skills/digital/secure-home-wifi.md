SECURE YOUR HOME WIFI

Steps:
1. ACCESS YOUR ROUTER: Open a browser. Go to 192.168.1.1 or 192.168.0.1 (check your router's label for the exact address and default login). Some routers use an app instead.
2. CHANGE THE ADMIN PASSWORD: The default admin password is public knowledge. Change it to something strong and unique. Store it in your password manager.
3. CHANGE THE WIFI PASSWORD: Use a strong password (12+ characters). Share it with guests verbally, not printed on a sticker.
4. SET ENCRYPTION TO WPA3: If your router supports it. Otherwise, WPA2-AES. Never WEP or WPA (both are broken). Check under Wireless Security settings.
5. CHANGE THE DEFAULT NETWORK NAME (SSID): Don't include your name, address, or router model. Something generic is fine.
6. DISABLE WPS (Wi-Fi Protected Setup): It has known vulnerabilities. Found under wireless settings.
7. UPDATE ROUTER FIRMWARE: Check for updates in the router admin panel. Enable auto-update if available.
8. OPTIONAL — SET UP A GUEST NETWORK: Most modern routers support this. Keeps visitors off your main network. Use a different password.

Notes:
- If your router is more than 5 years old, consider replacing it. Older routers may not receive security updates.
- If you see unknown devices on your network: change the wifi password immediately and reconnect only your devices.
- Disable remote management unless you specifically need it.
